# 이음(理音) 클릭형 프로토타입

선부3동 지역사회보장협의체를 위한 모바일 우선 제품 검증용 프로토타입입니다.

## 체험 역할

- **후원자:** 사진 여러 장이 포함된 나눔 등록, 매칭 요청, 전달 기록
- **협의체 위원:** 비식별 케이스 등록, 사진·썸네일, 케이스별 `등록→검토→자원→제안→승인→전달→완료` 진행 표시, 케이스에 맞는 후원 나눔 연결, 접이식 검색·필터, 조건 일치 이유, 수량·전달일·담당자 매칭 제안, 승인 상태 추적
- **운영자:** 긴급 케이스, 매칭 적정성·이해충돌 검토, 서로 다른 승인자 2인의 공동 승인, 최종 승인 시 재고 예약, 조직·보고 관리

## 전체 프로세스 한눈에 보기

첫 화면의 `전체 프로세스 한눈에 보기`에서 다음 내용을 한 페이지로 확인할 수 있습니다.

- 필요 발굴부터 완료 보고까지의 7단계 역할별 흐름
- 일반 케이스
- 긴급 지원
- 수정·재검토
- 부분 전달·재매칭

## 개인정보 원칙

- 대상자는 앱 계정을 만들지 않습니다.
- 후원자에게 실명·전화번호·상세 주소를 노출하지 않습니다.
- 개인 위원 단독 결정이 아닌 협의체 승인 흐름을 사용합니다.

> 이 저장소는 실제 개인정보를 사용하지 않는 데모 프로토타입입니다.
