# 이음(理音) 클릭형 프로토타입

선부3동 지역사회보장협의체를 위한 모바일 우선 제품 검증용 프로토타입입니다.

## 체험 역할

- **후원자:** 사진 여러 장이 포함된 나눔 등록, 로그인한 후원처의 나눔·매칭·완료 기록을 왼쪽 썸네일과 오른쪽 핵심 정보의 연속 피드로 확인
- **협의체 위원:** 비식별 케이스 등록, 등록 직후 유사 후원 나눔 자동 추천, 후원 목록을 왼쪽 썸네일·오른쪽 핵심 정보의 연속 스크롤 피드로 확인, 전체·식사·물품·서비스 카테고리별 최신순 탐색, 후원 나눔 선택 후 미매칭 케이스 선택, 수량·전달일·담당자 매칭 신청, 전체 진행 단계 추적
- **운영자:** 케이스 검토·배송 목록은 케이스 썸네일 피드로, 후원 등록 검토는 후원 이미지 피드로 확인, 매칭 적정성·이해충돌 검토, 서로 다른 승인자 2인의 공동 승인, 최종 승인 시 재고 예약, `배송 대기→배송 중→완료` 기록, 조직·보고 관리

## 매칭 흐름

1. **케이스 등록에서 바로 연결:** 케이스의 지원 종류·수량·기한을 기준으로 유사한 후원 물품·서비스를 자동 추천하고, 위원이 선택해 매칭 신청으로 이어갑니다.
2. **연결 탭에서 수동 연결:** `연결 대기` 상태의 후원 물품·서비스만 최신 등록순으로 표시합니다. 전체·식사·물품·서비스 카테고리에서 품목을 누르면 큰 이미지, 수량, 기한, 전달 방식, 지역과 설명을 확인하는 상세 화면이 열립니다. 상세 화면의 `내 케이스에 연결`을 누른 뒤 아직 매칭되지 않은 케이스를 선택합니다. 매칭 신청한 자원은 운영 검토가 끝날 때까지 연결 목록에서 제외됩니다.
3. **운영 처리:** 운영 검토와 공동 승인 2인을 거쳐 재고를 예약하고 배송 대기 업무를 생성합니다. 이후 배송 시작과 전달 완료를 기록합니다.

## 전체 프로세스 한눈에 보기

첫 화면의 `전체 프로세스 한눈에 보기`에서 다음 내용을 한 페이지로 확인할 수 있습니다.

- 필요 발굴부터 완료 보고까지의 7단계 역할별 흐름
- 일반 케이스
- 긴급 지원
- 수정·재검토
- 부분 전달·재매칭

## 개인정보 원칙

- 대상자는 앱 계정을 만들지 않습니다.
- 후원자에게 실명·전화번호·상세 주소를 노출하지 않습니다.
- 개인 위원 단독 결정이 아닌 협의체 승인 흐름을 사용합니다.

> 이 저장소는 실제 개인정보를 사용하지 않는 데모 프로토타입입니다.
